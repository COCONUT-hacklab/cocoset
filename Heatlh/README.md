## Penggunaan

Dataset ini dapat digunakan untuk berbagai tujuan penelitian, termasuk:
* Klasifikasi penyakit
* Prediksi risiko kesehatan
* Analisis tren kesehatan masyarakat
* Penemuan pola dalam data medis, etc
